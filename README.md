This package provides a global CLI for [Pangolin Pattern Library framework](https://pangolinjs.org).


## Installation

```bash
# npm
npm install -g @pangolin/cli

# Yarn
yarn global add @pangolin/cli
```


## Create a new project

```bash
pangolin create project-name
```


## Contribute

```bash
yarn test
# or
npm test
```
